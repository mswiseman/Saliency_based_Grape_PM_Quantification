#### Quantification components


```console
- classification: Train classification models
- segmentation: Train segmentation models
- analysis: Analysis helper functions (masking)
- visualization: Saliency components
- sanity check: Sanity check components
- scripts: Shell script to run analyzer programs
```


```console
- patch_correlation.py: Run Pearson's correlation at the image patch level
- plot_sal_map_patch.py: Plot saliency maps of image patches
- leaf_correlation.py: Run Pearson's correlation at the leaf disk level
- plot_sal_map_leaf.py: Plot saliency maps of leaf disk samples
```
